This plugin makes it mandatory to be logged in before viewing any content.

== Description ==

If you need to make your Wordpress site private, this plugin makes it possible for 
your users having to log in before being able to view any content on the website.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wprequireauth.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
